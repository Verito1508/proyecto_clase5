Clase 5, como subir un proyecto a github

Pasos en local, como crear un proyecto

  1. crear una carpeta
  2. crear una carpeta del proyecto mkdir "nombre_carpeta"
  3. entrar a la carpeta del proyecto cd "nombre_carpeta"
  4. creamos un archivo index.html touch index.html
  5. iniciar un repositorio github

Pasos en github, como crear un nuevo repositorio

  1. entrar a github
  2. crear un nuevo repositorio
  3. copiar la url del repositorio
  4. volver a la terminal
